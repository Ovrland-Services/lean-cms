# LeanCMS

A lightweight Rails CMS with in-context editing for marketing sites.

**Documentation:** [leancms.dev](https://leancms.dev)

## Installation

Add to your Gemfile:

```ruby
gem "lean_cms"
```

Run the installer:

```bash
bundle install
rails generate lean_cms:install
rails db:migrate
```

## Quick start

See the full [Getting Started guide](https://leancms.dev/docs/getting-started) or install the interactive demo:

```bash
rails generate lean_cms:demo
```
